### Collections Framework

![Collections Framework](./img/collections-framework(1).png)

#### Map
검색할 수 있는 인터페이스로 **데이터 삽입 시, (Key, Value) 형태로 삽입된다.**

#### Collection

* **List**
순서가 있는 Collection으로 데이터 중복이 가능하다.

* **Set**
집합적인 개념의 Collection으로 순서의 의미가 없고, 데이터 중복이 불가능하다.


#### Collections Framework는 어떻게 선택할까?

1. **Map과 Collection 인터페이스 중 선택**

2. **목적에 따라 Map, List, Set 각각의 하위 구현체를 선택**

```
Map: HashMap, LinkedHashMap, HashTable, TreeMap
List: LinkedList, ArrayList
Set: TreeSet, HashSet
```
